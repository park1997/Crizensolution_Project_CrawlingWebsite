# CrizenSolution_CrawlingWebsite
크리젠솔루션 인턴기간중에 완료했던 프로젝트입니다.
동적웹페이지인 네이버부동산을 Selenium과 Jsoup을 이용하여 필요한 정보를 크롤링하여 동적테이블로 정리하여 웹으로 로컬에 구현했습니다.
동적테이블은 html에서 javascript로 구현하였습니다.

"리얼 서치" 라는 웹사이트의 구현의 일부분을 기여하였습니다.
